# Yunzai-Bot v3(续航版)
[![docker test](https://github.com/Lycofuture/Yunzai-Bot/actions/workflows/Docker%20Build%20Test.yml/badge.svg?branch=dev&event=push)](https://github.com/Lycofuture/Yunzai-Bot/actions/workflows/Docker%20Build%20Test.yml)

云崽v3.0，原神qq群机器人，通过米游社接口，查询原神游戏信息，快速生成图片返回

项目仅供学习交流使用，严禁用于任何商业用途和非法行为

[原版](https://github.com/Le-niao/Yunzai-Bot) && [目前功能](https://github.com/Lycofuture/Yunzai-Bot/tree/main/plugins/genshin)

## 使用方法

>环境准备： Windows or Linux，Node.js（[版本至少v16以上](http://nodejs.cn/download/)），[Redis](https://redis.io/docs/getting-started/installation/)

参考[安装教程](https://github.com/Lycofuture/Yunzai-Bot/wiki/%E5%AE%89%E8%A3%85%E6%95%99%E7%A8%8B)进行安装

如果上面的点不开可[点击此处](https://gitee.com/lycofuture/Yunzai-Bot/wikis/安装教程)
